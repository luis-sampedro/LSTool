Project tools

These are the four tools I want to develop inside this space. Some are already prototyped, others are still work in progress.

  • LS Data Lab (LSDL)
  • LS Auto Leech (LSAL)
  • LS Current App (LSCA)
  • LS Design Tools (LSD)

LS Data Lab (LSDL)

Online workspace for processing nautical data: GPS logs, sensor streams and race tracks. Inspired by tools like Sailtrac, focused on speed, VMG and manoeuvre analysis.

Status: prototype · Click to open

LS Auto Leech (LSAL)

Automated data collection engine. The idea is to “leech” structured information from different sources (race trackers, weather APIs, onboard systems) into a clean database for later analysis.

Status: concept · UI placeholder

LS Current App (LSCA)

Tool to visualise currents, wind and boat performance together. Designed to help understand how external conditions affect routing, laylines and tactical decisions on the racecourse.

Status: concept · UI placeholder

LS Design Tools (LSD)

A collection of small utilities for yacht design: hull and foil parameter explorers, basic VPP-style calculators and links with CFD workflows.

Status: concept · UI placeholder
